# Laptop Price Scraper

## Business Problem
E-commerce businesses need to monitor competitor pricing across hundreds of products. Manual checking is time-consuming and error-prone.

## What It Does
Scrapes all laptop listings from an e-commerce site and extracts:
- Product name
- Price
- Star rating
- Number of reviews

## Output
Clean Excel file ready for analysis or reporting.

## Tech Stack
Python, BeautifulSoup, Requests, Pandas
